What is CBS Bot?

CBS Bot keeps chats warm (welcome messages and subtle replies), shares occasional inspiring messages, lets admins configure group links, can block unwanted bots, supports raid mode for Twitter/X engagement, and offers an invite contest with personal links and a leaderboard.

Key features

  • 👋 Auto-welcome new members (customizable)
  • 💬 Subtle replies to greetings / thanks / fun max 1/30s per chat
  • ✨ Auto-chat: inspiring messages every 30 min (on by default)
  • 🛡️ Antibot radar: auto-block new bots (default off) + whitelist
  • 🚩 Raid mode: post a raid message with /raid
  • 📊 Raid Counters: Like / Repost / Comment buttons with live tally
  • 🧮 Join-raid status: monitor burst joins in a time window
  • 🏆 Invite contest: personal invite links, auto scoring, leaderboard
  • 🔗 Group links: website, twitter, telegram, discord, facebook, instagram, youtube, mint, ca
  • 🔧 Admin-only configuration

Permissions checklist

  • Read & send messages
  • Antibot: Ban users
  • Invite Contest: Create invite links

In BotFather → Group Privacy = OFF so the bot can see messages.

Raid Mode — Posting & Counters

Setup

  • /set raid on — enable raid mode
  • /raidtarget <url> — set Twitter/X URL (post, profile, hashtag)
  • /raidmsg <text with {url}> — set template; {url} will be replaced
  • /raid — post raid message with Like/Repost/Comment buttons
  • /raidnow — admin shortcut to post regardless of raid mode

Members can tap each button once; the bot counts unique users per action.

Engagement goals & progress

  • /raidsetreq <likes> <reposts> <comments> — set targets
    Example: /raidsetreq 50 40 30
  • /raidprogress — show current totals vs targets
  • /raidreset — reset counters for a new raid
  • /x — show the configured Twitter/X link
FAQ: “How do I know the raid is counting?”
Use /raidprogress. You’ll see e.g. Likes 12/50 | Reposts 9/40 | Comments 5/30.

Join-Raid Detector (anti-bot bursts)

Configure

  • /setraid <joins> <minutes> — threshold window
    Example: /setraid 10 5
  • /raidstatus — show recent joins vs threshold

How it works

Each new join timestamps into a rolling window per chat. /raidstatus shows the count in the last N minutes and your threshold. Use it to decide when to moderate stricter.

Troubleshooting & Ops

Systemd restart

  1. Update bot.py in ~/CBSbot
  2. sudo systemctl restart adje
  3. Logs: journalctl -u adje -f

Changed the service file? sudo systemctl daemon-reload then restart.

Common issues

  • Buttons not counting? Test with /raidsetreq 0 0 0 then /raid and tap once.
  • “How to set comment & repost?” /raidsetreq <likes> <reposts> <comments>
  • “Is it counting?” /raidprogress
  • Join peaks? /setraid 10 5 → /raidstatus
  • Invite link fails? Make bot Admin with “Invite users via link”.

Privacy & Data

  • No personal data is stored or profiled.
  • Settings persist until changed or the group is removed.
